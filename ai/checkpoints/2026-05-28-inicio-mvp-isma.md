# Checkpoint: Inicio MVP ISMA

Fecha: 2026-05-28

Se adapto la orientacion del proyecto a Archivo Historico Digital ISMA.

Decisiones iniciales:
- Frontend React + Material UI.
- Backend Spring Boot Java.
- MySQL como base de datos.
- Cloudinary como proveedor inicial de imagenes.
- Capa `ImageStorageService` para poder cambiar de proveedor.
- Mobile-first por uso esperado desde celulares.
